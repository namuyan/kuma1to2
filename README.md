kumacoin 1.0 to 2.0 swapper
====
Kuma1.0 to Kuma2.0 swap tool.

Specifications
----
* Python3.6
* Windows/Linux

How to use
----
```bash
git clone https://github.com/namuyan/kuma1to2
cd kuma1to2
pip3 install --user -r requirements.txt
python3 gui.py
```
